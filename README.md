# 🩺 SymptoScan  
### AI-Powered Symptom Based Disease Prediction & Doctor Consultation Platform

SymptoScan is a smart healthcare web application that helps users predict possible diseases based on selected symptoms using Machine Learning and connect with doctors through paid online consultation.

It combines *AI prediction, **real-time consultation, **secure payments, and **admin analytics* into one platform.

---

# 🚀 Live Features

✅ Symptom based disease prediction  
✅ Machine Learning model integration  
✅ Confidence score results  
✅ Doctor listing & consultation booking  
✅ Razorpay payment gateway  
✅ Chat consultation system  
✅ Automated first responses / bot support  
✅ Prediction history tracking  
✅ Admin dashboard with revenue stats  
✅ JWT Authentication  
✅ Role based dashboards (Patient / Doctor / Admin)

# 🤖 Machine Learning Workflow

1. User selects symptoms  
2. Data converted into feature vector  
3. Trained model predicts disease  
4. Confidence score generated  
5. Result shown to user  
6. Option to consult doctor
